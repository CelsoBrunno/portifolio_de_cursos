Brunno Rocha // Professsor de T.I.

Cursos Ministrados & Proficiência

  • Desenvolver algoritmos

    "Curso fundamental para a formação de qualquer programador. Focado no desenvolvimento do raciocínio lógico e na resolução de problemas através de algoritmos. A jornada de aprendizado começa com os conceitos mais básicos da programação e avança progressivamente até a introdução à Programação Orientada a Objetos, preparando o aluno para aprender qualquer linguagem de programação."

    Conceitos: Lógica Algoritmos Orientação a Objetos
  • Programador de sistemas em Python

    "Capacitação completa para o desenvolvimento de sistemas com Python. O curso abrange desde a sintaxe fundamental e lógica de programação até os paradigmas da Programação Orientada a Objetos (POO), permitindo a criação de código limpo e modular. Ao final, os alunos são introduzidos ao desenvolvimento web com a criação de servidores e APIs utilizando o micro-framework Flask."

    Stack: Python Flask POO
  • Criar e manter Banco de Dados

    "Curso prático focado na gestão de bancos de dados relacionais, utilizando o sistema MySQL, um dos mais populares do mercado. Os alunos aprendem a modelar, criar e manter bancos de dados, realizando consultas complexas com a linguagem SQL. A ferramenta gráfica MySQL Workbench é utilizada para facilitar a administração e o design do banco de dados."

    Stack: MySQL SQL Workbench
  • Desenvolver aplicações web

    "Formação essencial para a criação de páginas e aplicações web modernas. O curso aborda o trio fundamental do front-end: HTML para a estruturação do conteúdo, CSS para a estilização e design responsivo, e JavaScript para criar interatividade e dinamismo nas páginas, resultando em interfaces de usuário ricas e funcionais."

    Stack: HTML5 CSS3 JavaScript
  • Desenvolver aplicações mobile

    "Curso imersivo no desenvolvimento de aplicativos para a plataforma Android. Utilizando a linguagem oficial Kotlin e o ambiente de desenvolvimento Android Studio, os alunos aprendem a construir interfaces de usuário tanto pelo método tradicional com XML quanto pela abordagem moderna com Jetpack Compose. O curso cobre os principais componentes do ecossistema Android para a criação de aplicativos nativos."

    Stack: Kotlin Android Studio Jetpack Compose
